# Nuthalapati Navya – Portfolio2

This is my personal portfolio website built with HTML, CSS, and a bit of JavaScript.  
It showcases my profile, skills, projects, and contact details as an aspiring Frontend & Software Developer.

## Features

- Clean single-page layout: **About, Skills, Certifications, Projects, Contact**
- **Responsive design** for desktop and mobile screens
- **Light / Dark mode toggle** (theme preference saved using `localStorage`)
- Project cards with:
  - “Open Project” buttons (live demo links)
  - “View Code” buttons (GitHub repositories)
- Contact section with quick links to **Email** and **LinkedIn**

## Tech Stack

- **HTML5** – structure and sections  
- **CSS3** – layout, responsive design, light/dark themes  
- **JavaScript (Vanilla JS)** – theme toggle and basic interactivity
